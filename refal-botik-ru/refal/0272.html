Числа, рассказывающие друг про друга

По адресу
http://www.refal.net/~korlukov/pearls/
в разделе "Рекурсивные числа" ранее был помещен материал об экспериментах по суперкомпиляции задачи о числах, которые несут информацию о себе.

Например, в числе 6210001000 первая цифра равна количеству нулей в нем самом, вторая цифра равна количеству единиц, и так далее, десятая цифра равна количеству девяток в числе.

Оказывается, имеются числа, рассказывающие аналогичным образом друг про друга.

Например, два числа 7101001000 и 6300000100.

Пример трех чисел состоит из трех 7-значных чисел
4110100 ; 4102000 ; 3300100 .

(первое число говорит, что во втором числе есть 4 нуля, 1 единица, 1 двойка, 0 троек, 1 четверка, 0 пятерок, 0 шестерок. Аналогично, второе число рассказывает про третье, а третье - про первое).

По адресу

http://www.refal.net/~korlukov/numbers.doc

я разместил статью школьника 11 класса из Гродно, в которой дается полное решение задачи.

Никаких экспериментов по суперкомпиляции этой задачи пока не проводилось.
