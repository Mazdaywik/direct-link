         Об'явление.

    На страничке http://www.botik.ru/pub/local/scp/refal5/
           ( синоним:
 http://www.botik.ru/pub/local/scp/refal5/refal5.html )
    выложена новая версия SCP4.

 Новости:
-----------------------------------------------------------------
   - Исправлено несколько ошибок.
     Большое спасибо Александру Корлюкову , за пересланные мне тесты на
ошибки.

   - Добавлено несколько новых демонстрационных примеров.
     Большое спасибо Александру Корлюкову за интересные примеры.

   - расширен входной язык описания программ для суперкомпиляции:
             -- допускаются открытые е-переменные в левой части
предложения;

             -- допускаются вызовы внутренних функций в левых частях
предложений;

             -- допускаются блоки;

             -- условия ( запятые ) в левых частях допускаются без каких
либо
                  ограничений;

             -- входной язык отныне содержит в себе полный одномодульный
                  Рефал-5!

             Я благодарен Виктору Кистлярову, который убедил меня в
             необходимости расширения входного языка до полного Рефала.

       ( подробности см. в DOC/USE.TXT, DOC/INREF4.TXT )

   - расширен входной язык описания задач на суперкомпиляцию:
                - допускается описание нескольких MST-схем в одном
MST-файле,
                   такое описание позволяет решать несколько задач (по числу
схем)
                   одновременно. С точки зрения программы-ответа это
неравносильно
                   последовательному решению данных задач;

     ( подробности см. в DOC/MST.TXT )

  - реализована новая "функция"-разметка-подсказка Appl__ .
    Эта функция должна быть определена на Рефале как тождественная.
    Её вызовы меняют развитие стека во время "ленивой" суперкомпиляции.
    Вызов этой функции делает полную декомпазицию функциональной
    структуры своего аргумента. Т.е. форсирует одношаговое вычисление
    самого внутреннего , самого левого вызова в своём аргументе.

     ( подробности см. в DOC/USE.TXT, DOC/INREF4.TXT )

  - Большое спасибо всем пользователям Scp4.

 Андрей Немытых.
