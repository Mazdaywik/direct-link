Привет!

В eDonkey2000 поддерживается "список фальшивок", который содержит хеш-коды
файлов, которые содержат не то, что сказано в их имени.

Работа с этим списком поддерживается в eMule Plus. Можно закачать список
фальшивок, и тогда Мульчик будет предупреждать о фальшивках прямо при показе
результатов поиска (в колонке Fake Check).

Чтобы это заработало, идем в Preferences->General и находим опцию Update
fake list from this URL. Рядом с этим текстом находится текстовое окошко, в
котором должен быть, например, следующий URL

http://donkeyfakes.gambri.net/public/ed2kfakes.txt

После этого нажимаем кнопку Update. После этого из сети eDonkey
автоматически выкачивается файл fakes.rar, содержимое которого автоматически
(после окончания закачки) инсталируется куда надо.

После этого, если остро стоит проблема исходящего трафика, файл fakes.rar
можно удалить.
