Dear Leonid,

Refal+ scripting support has been included into recent release of refal+.
There exist several issues, mainly related to security, but it is already
usable.

To use RFPS:

# mkdir /var/spool/rfps
# chmod 777 /var/spool/rfps

Then, just put the string
#!/usr/bin/rfps

in the beginning of a script (not just CGI) :-)

An example of rfps usage for CGI scripting is at
http://shade.msu.ru/Serge/reftst/

Sincerely yours,
